    `name: Security Audit`. `on:` — `schedule: [{ cron: '0 9 * * *' }]` (daily, was weekly),
    `push: { branches: [main] }`, and `workflow_dispatch:`.

    Job `osv-scan` — call the reusable workflow
    `uses: google/osv-scanner-action/.github/workflows/osv-scanner-reusable.yml@v1`
    with `scan-args: '--lockfile pnpm-lock.yaml'`.

    Job `license-check` — `runs-on: ubuntu-latest`; steps: `actions/checkout@v4`,
    `pnpm/action-setup@v4`, `actions/setup-node@v4` (node 22), `pnpm install --frozen-lockfile`,
    then a "Check license allowlist" step running:
    ```
    pnpm exec license-checker-rseidelsohn \
      --excludePrivatePackages \
      --failOn "GPL;AGPL;LGPL;CC-BY-SA;OSL" \
      --onlyAllow "MIT;Apache-2.0;BSD-2-Clause;BSD-3-Clause;ISC;0BSD;BlueOak-1.0.0;CC0-1.0;Python-2.0;Unlicense;MPL-2.0"
    ```
    The `--excludePrivatePackages` flag is required (Pitfall 5 — internal `@spatula/*`
    workspace packages would otherwise fail the allowlist).

    Job `secret-scan` — `runs-on: ubuntu-latest`; steps:
    `actions/checkout@v4` with `fetch-depth: 0` (full history required for trufflehog),
    `gitleaks/gitleaks-action@v2` with `env: { GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }} }`,
    and `trufflesecurity/trufflehog@main` with `path: ./`, `base: ''`, `head: HEAD`,
    `extra_args: --only-verified`.

  </action>
