<objective>
Ship `render.yaml` at the repo root (DEPLOY-02) so a fresh clone deploys the full stack on a Render free-tier account, reachable on the assigned URL (SC#2). Because Render's free tier has no Background Worker type (D-06), the blueprint runs the worker IN-PROCESS with the API on one free Web Service via `SPATULA_EMBEDDED_WORKER=1` (the shim built in Plan 01). Purpose: Zero-external-setup, one-click try-it deployment on a genuine free tier (SC#2).
Output: render.yaml + docs/runbooks/render-deploy.md.

This plan DEPENDS on Plan 01 (embedded-worker mode + apps/api/dist/main.js bootstrap). The Render startCommand targets the real bootstrap.
</objective>
