    Create `apps/api/src/lib/cors-origin.ts` exporting `buildOriginMatcher(raw: string): { match(origin: string): boolean } | null` and a thin `corsOriginFn` factory. Implement per D-08 / RESEARCH Pattern 3:

    - Split `raw` on `,`, trim each part, drop empties.
    - If the result is empty → return `null` (caller fails boot).
    - For each part: if it contains `*`:
      - reject a bare `*` (D-10: explicit "no `*` allowed" rule) — if any part is exactly `*`, return `null`.
      - compile a single-label wildcard: escape regex metacharacters EXCEPT `*`, then replace `\*` with `[^./]+` (exactly one subdomain label — no dots, no slashes; this prevents `*.spatula.dev` from matching `foo.bar.spatula.dev` or `evil.spatula.dev.attacker.com`). Wrap as `^...$`. Concrete: `https://*.spatula.dev` → `/^https:\/\/[^./]+\.spatula\.dev$/`.
    - else: add to an exact-match `Set<string>`.
    - `match(origin)` returns true if `exact.has(origin)` OR any pattern `.test(origin)`.

    Then rewrite the CORS block in `apps/api/src/app.ts`:
    1. `const matcher = buildOriginMatcher(getEnvOrDefault('CORS_ALLOWED_ORIGINS', 'http://localhost:3000'))`.
    2. `if (!matcher) throw new Error('CORS_CONFIG_INVALID: CORS_ALLOWED_ORIGINS is empty or malformed (a bare "*" is not allowed)')` — boot-time panic, a plain `Error` NOT a `SpatulaError` (RESEARCH Open Question 1 — boot failures are not HTTP responses).
    3. `cors({ origin: (origin) => (origin && matcher.match(origin) ? origin : null), ... })` — return the echoed origin string on match, `null` on no-match (Hono treats null as block).
    4. Extend `exposeHeaders` to exactly `['X-RateLimit-Limit', 'X-RateLimit-Remaining', 'X-RateLimit-Reset', 'X-Request-Id', 'Retry-After']` (D-09 — adds `X-RateLimit-Reset` + `Retry-After`).
    5. Keep `maxAge: 86400`, `credentials: true`, `allowMethods`, `allowHeaders` unchanged (D-09 — preflight cache stays at 86400s).
